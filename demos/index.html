SlideMap demos fixed width examples

Horizontal slidemap

A very simple example. Click the areas on the map to move the slide.

3D slidemap

More elavorated example. Multiple areas with diferent horizonta and vertical position. Plus, to add more focus on the region, we can specify a zoom with the fifth parameter of the data-coords. The unit of this value are pixels and a positive value means zoom in while a negative value means zoom out.

slidemap with captions

You can also add a caption to each area. A caption is a div element where you can place whatever you want to show for an specific action.
